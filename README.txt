You can start the game by writing "java Test" to terminal.

To save and load, write the file name in the text field next to the button and then click the button.
save1.txt is player vs. computer game
save2.txt is player vs. player game

all buttons works even if the game has ended; so you can undo, reset and resize after finishing a game

To resize the board write the new size of the board to text field next to the "Board Size:" and then press enter.

You can close display messages (win message or error messages) by pressing OK button inside of the frame.

Source codes are in the HEXGUI/codes path in current folder.
